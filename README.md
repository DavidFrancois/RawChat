# RawChat
A Raw Chat app

## Getting Started

To get the repo :

```
git clone git@github.com:DavidFrancois/RawChat.git
```

### Prerequisites

NodeJS v6.xx

npm v3.xx

Unix System

### Installing

To install npm dependencies use the script at the root of the repo

``` bash
sh setup.sh
```

## Deployment

To launch the project you have to launch both the client and server

``` bash
cd ./server
node index.js
```

``` bash
cd ./client
ng serve --open
```
